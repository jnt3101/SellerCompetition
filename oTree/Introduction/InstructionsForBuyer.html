Die Lotterien

In dieser Studie könnende Teilnehmende Lotterien kaufen, die von anderen Teilnehmenden in der Studie angeboten werden. Sowohl Käufer als auch Verkäufer können einen zusätzlichen Bonus verdienen. Die mögliche Bonushöhe hängt von den Entscheidungen in der Studie ab, zum Beispiel von den gewählten Preisen für Lotterien, ob die Lotterie gekauft wird und ggf. vom Ergebnis einer zufälligen Ziehung. Der Bonus jedes Teilnehmenden wird daher sowohl von eigenen Entscheidungen, als auch von Entscheidungen der anderen Teilnehmenden beeinflusst.

Unten sehen Sie ein Beispiel für eine Lotterie:

Lotterie
Wahrscheinlichkeit 40 % 30 % 30 %
Auszahlung 20 Münzen 40 Münzen 60 Münzen

In diesem Beispiel zahlt die Lotterie entweder 20 Münzen mit 40 % Wahrscheinlichkeit, oder 40 Münzen mit 30 % Wahrscheinlichkeit, oder 60 Münzen mit 30 % Wahrscheinlichkeit. Wichtig ist: Eine Lotterie zahlt immer nur einen der Beträge aus; sie zahlt also entweder 20 Münzen oder 40 Münzen oder 60 Münzen.


Ihre Rolle

Sie wurden der Rolle eines Käufers zugeordnet. In diesem Teil der Studie werden Ihnen Lotterieangebote gezeigt, die von anderen Teilnehmenden gestaltet wurden. Ihre Aufgabe ist es, diese Lotterien zu beurteilen, Ihre Einschätzungen zu ihnen abzugeben und zu entscheiden, ob Sie eine der Lotterien kaufen möchten.


Ihre Entscheidungen
{% if group.group_type == "1S1B" %}

In Ihrer Gruppe gibt es einen Verkäufer und einen Käufer. Sie sehen daher eine Lotterie pro Runde, die von einer anderen Person in der Studie angeboten wird.

Für diese Lotterie fragen wir Sie zunächst, wie oft Sie glauben, dass Sie den höchsten Gewinn dieser Lotterie erhalten würden, wenn Sie die Lotterie 100 Mal spielen würden. Dies ist einfach Ihre persönliche Einschätzung, wie häufig die höchste Auszahlung gezogen werden würde.

Nachdem Sie Ihre Einschätzung abgegeben haben, entscheiden Sie, ob Sie diese Lotterie zu dem Preis kaufen möchten, den der Verkäufer festgelegt hat. Sie können entweder die Lotterie kaufen oder sich entscheiden, sie nicht zu kaufen.

{% else %}

In Ihrer Gruppe gibt es einen Käufer (Sie) und zwei Verkäufer. Das bedeutet, dass Sie mehr als eine Lotterie pro Runde sehen werden. Nach jeder Runde wird eine neue Gruppe mit zwei zufälligen Verkäufern gebildet. Das heißt, dass Sie jede Runde auf andere Verkäufer treffen können.

Für jede dieser Lotterien fragen wir Sie zunächst, wie oft Sie glauben, dass Sie den höchsten Gewinn erhalten würden, wenn Sie die jeweilige Lotterie 100 Mal spielen würden. Dies ist einfach Ihre persönliche Einschätzung, wie häufig die höchste Auszahlung gezogen werden würde.

Nachdem Sie Ihre Einschätzungen abgegeben haben, entscheiden Sie, ob Sie eine Lotterie kaufen möchten und, falls ja, welche. Sie dürfen maximal eine Lotterie kaufen. Das bedeutet, Sie können:

  • die Lotterie von Verkäufer 1 kaufen,
  • die Lotterie von Verkäufer 2 kaufen,
  • oder sich dafür entscheiden, keine Lotterie zu kaufen.

Die Bonuszahlung

Insgesamt wird es in der Studie {{ C.N_LOTTERIES }} Runden geben. Sie können in jeder Runde zusätzliche Münzen verdienen. Wenn Sie sich für die Details interessieren, finden Sie unten auf der Seite eine ausführliche Beschreibung des Bonusmechanismus.

Ihre konkrete Bonuszahlung hängt von Ihren Entscheidungen ab:

  • Wenn Sie eine Lotterie kaufen, wird per Computer eine zufällige Auszahlung dieser Lotterie gezogen, und diese Auszahlung bestimmt Ihre Bonuszahlung.
  • Wenn Sie sich entscheiden keine Lotterie zu kaufen, entspricht Ihre Bonuszahlung stattdessen dem niedrigeren Preis der beiden Lotterien, die in Ihrer Gruppe angeboten wurden.

Während des Experiments verwenden wir eine fiktive Währung, die wir „Münzen“ nennen. Für die Bonuszahlung werden alle Münzen in Euro umgerechnet zum Kurs von {{ C.EXCHANGE_RATE }} Münzen = 1 Euro.
